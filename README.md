# destboard-client

## Description

The web app for [step63r/destboard](https://github.com/step63r/destboard).

## Requirement

- Node.js v16.15.1
- React 5.0.1
- TypeScript 4.9.4

## Usage

Run script as below.

```
$ npx react-scripts start
```

## Install

Fork and clone this repository.

```
$ git clone git@github.com:[yourname]/destboard-client.git
```

Install packages.

```
$ npm ci
```
